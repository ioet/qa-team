# Cypress Framework

Cypress framework is a JavaScript-based end-to-end testing framework built on top of Mocha and Chai

# Requirements

- [Nodejs](https://nodejs.org/es/)

## Installation

```bash
npm install
```

## Run tests

```bash
npm run cypress:open
```
